EasyFocus

Web App

Dez 2025

O EasyFocus é um projeto acadêmico desenvolvido com o objetivo de aplicar, na prática, conceitos fundamentais de desenvolvimento web utilizando HTML, CSS e JavaScript. A aplicação serviu como base para consolidar conhecimentos teóricos por meio da construção de uma solução funcional e voltada à usabilidade.

Durante o desenvolvimento, foi identificada a necessidade de uma forma clara e intuitiva de relacionar tarefas com datas. A ausência dessa integração dificultava o planejamento e o acompanhamento das atividades, impactando a experiência do usuário e a organização das informações.

Durante o desenvolvimento do EasyFocus, o projeto se mostrou fundamental para a consolidação dos principais conceitos de aplicações web, como estruturação de páginas, estilização, manipulação do DOM e organização da lógica em JavaScript. Além do aspecto técnico, o processo contribuiu diretamente para o desenvolvimento de boas práticas de código e entendimento do funcionamento de aplicações no navegador.

O trabalho foi realizado de forma colaborativa em equipe, utilizando um quadro Kanban para organização das tarefas e aplicando conceitos inspirados no Scrum, como divisão de responsabilidades, acompanhamento de progresso e entregas incrementais. Essa dinâmica fortaleceu a comunicação entre o time e proporcionou uma experiência próxima à realidade de projetos de desenvolvimento de software.

Tecnologias Utilizadas

Front-end: HTML, CSS, JavaScript

Ferramentas: Git, Kanban
