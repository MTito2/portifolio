Flash Gen

Desktop App

Set 2025

O projeto consiste em uma aplicação desenvolvida em Python para automatizar a criação de flashcards com apoio de inteligência artificial, facilitando o processo de estudo utilizando a técnica de repetição espaçada. A solução foi pensada para otimizar a criação de materiais de estudo e reduzir o esforço manual envolvido nesse processo.

A necessidade surgiu durante o início dos estudos de inglês, quando ficou evidente que a criação manual de flashcards pode ser lenta e repetitiva, impactando a constância do aprendizado. As ferramentas existentes não atendiam completamente às necessidades do projeto ou possuíam um custo elevado, o que motivou o desenvolvimento de uma solução personalizada.

A aplicação permite que o usuário informe palavras ou frases que deseja aprender. A partir desse input, um agente de IA gera automaticamente o conteúdo dos flashcards, incluindo tradução, exemplos de uso, variações e outras informações relevantes. Quando possível, o sistema também associa imagens ao conteúdo, enriquecendo o material visual dos cartões.

O projeto foi desenvolvido em Python e integra a OpenAI API para geração do conteúdo dos flashcards, a Unsplash API para obtenção de imagens e add-ons do Anki para automatizar a criação e inserção dos cards no deck. Essa integração permitiu construir um fluxo totalmente automatizado, desde a entrada dos dados até a visualização final no Anki, consolidando conhecimentos em APIs, automação e integração entre serviços externos.

Tecnologias Utilizadas

Back-end: Python

Ferramentas: Git, OpenAI API, Unsplash API
