Focus Timer

Desktop App

Dez 2025

Focus Timer é um cronômetro automático que monitora as telas ativas e o movimento do mouse, identificando quando o usuário está focado, realizando outras atividades ou ocioso. A ideia surgiu devido à minha dificuldade em marcar o tempo de foco enquanto realizava outras tarefas. Com o cronômetro manual, eu frequentemente esquecia de pausar ou retomar a contagem.

Com o Focus Timer, não preciso me preocupar com isso: a aplicação pausa e retoma automaticamente e ainda gera um relatório mensal, que serve para análise e como forma de motivação para manter o foco em estudos, projetos ou qualquer outra atividade que eu queira monitorar.

Com esse projeto, aprendi a integrar o frontend com o backend de forma organizada e funcional, além de praticar Programação Orientada a Objetos na criação do cronômetro e dos monitores no backend. Também aprendi a utilizar um banco de dados NoSQL para persistência dos dados, permitindo o uso da aplicação em diferentes computadores com os dados sincronizados.

Foi utilizada a biblioteca PyWebView, do Python, que permite a criação de interfaces utilizando HTML, CSS e JavaScript. Para a persistência dos dados, foi utilizado o Firebase como banco de dados, responsável por armazenar as informações de tempo de foco, tempo ocioso, tempo bruto e a descrição das atividades.

Tecnologias Utilizadas

Front-end: HTML, CSS, JavaScript

Back-end: Python, PywebView

Ferramentas: Git

Banco de dados: Firebase
